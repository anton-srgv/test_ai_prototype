# Test AI — UI Prototype

Interactive HTML prototype for the Test AI project screen redesign.

## Features
- Linear-style test case list with sections
- Status system: Новый → Генерация → Доработка → Готов
- Context menu on row actions (⋮)
- Right drawer with test case info
- Empty editor page with breadcrumb navigation
- Inline SVG Lucide icons
- Golos Text font

## How to view
Open `index.html` in a browser. No build step required.

## Design references
- Linear (list layout, density, hover patterns)
- Vercel (typography, monochrome palette)
- Notion (warmth, breathing room)
- shadcn/ui (tabs, buttons, filter chips)
- Retool (editor — future)

## Status
Concept prototype for CEO/PM presentation. Not production code.
